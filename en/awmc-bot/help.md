# Bind BOT
1. Before using any maiBot features, we recommend binding it with the `maibind` command.

2. Enter the above command in the group and follow the prompts.

# Upload b50
1. When using the b50 upload feature, ensure you have already bound either Diving-Fish or Lxns.

To bind Diving-Fish, use `maibindfish`; to bind Lxns, use `maibindlx`.

Diving-Fish requires an import token: http://diving-fish.com/maimaidx/prober/
Lxns requires a friend code: https://maimai.lxns.net/user/profile

![diving-fish-1.png](/images/diving-fish-1.png)
![lx-1.png](/images/lx-1.png)


Ensure permissions are enabled. For Lxns, allow all access in the privacy settings at https://maimai.lxns.net/user/settings.



2. Enter `maiu` to upload to Diving-Fish, `maiul` to upload to Lxns, `maiua` to upload to both.



FAQ
1. User not found during query

Ensure the correct QQ number is entered in Diving-Fish's profile.

2. Upload shows login failure

Account is still logged in on maimaiDX or in the 15 minute temp ban.
